Successfully running on {host}, now you have to append the following to your SSH config file (~/.ssh/config):

Host *.trycloudflare.com HostName %h User root Port 22 ProxyCommand access ssh --hostname %h

To connect using your terminal, type this command:

ssh {host}

[Optional] You can also connect with VSCode Remote SSH (Ctrl+Shift+P and type "Connect to Host..."). Then, paste this hostname

{host}
